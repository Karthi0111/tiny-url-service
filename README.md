# Tiny URL Service

## Overview

This project aims to design and implement a Tiny URL service that converts long URLs into shorter URLs and redirects users to the original URLs.

## Objectives

- Generate short URLs
- Store URL mappings
- Redirect users
- Handle collisions
- Follow software engineering principles

## Technology Stack

- Python
- Flask
- SQLite/PostgreSQL
- SHA-256 Hashing

## Project Status

Phase 1: System Design Documentation
Phase 2: Python Implementation
